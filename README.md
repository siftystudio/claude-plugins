# Sifty Claude Code Plugins

A self-hosted [Claude Code](https://claude.ai/claude-code) plugin marketplace, serving as a directory of plugins associated with our software offerings.

## Setup

Before you can install a Sifty plugin, you'll need to register our marketplace:

* Boot up Claude Code: `claude`.
* Inside your Claude Code session, add our marketplace:  `/plugin marketplace add siftystudio/claude-plugins`.

Now you'll be able to install a plugin using its command from in the table below.

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| [`viv`](https://github.com/siftystudio/viv/blob/main/plugins/claude) | Author and developer support for the Viv engine for emergent narrative. | `/plugin install viv@siftystudio` |
